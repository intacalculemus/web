.calcuLEMUs
El término calcuLEMUs se compone de dos palabras: 
-  "calculus" del latín que significa "pequeña piedra utilizada para el recuento", 
- y el término "lemu" que significa bosque o arboleda en lengua Mapuche.
 
El objetivo de este desarrollo es facilitar el procesamiento de datos de inventario forestales, tiene una interfase simple y desde la consola principal se accede a tres tipos de operaciones:
 
- En primer lugar el procesamiento de datos de inventarios de plantaciones forestales,
- en segundo lugar se puede realizar el procesamiento de datos de inventario de bosque nativo, 
- y en tercer lugar se puede simular el crecimiento y estimación de producción de plantaciones forestales utilizando el simulador Piltriquitrón.
 
Cada uno de los detalles necesarios para llevar adelante los cálculos se explicarán en las correspondientes secciones. Para esto se han creado diagramas de flujo de acciones que realiza el calculador para mejor la comprensión por parte del usuario.
