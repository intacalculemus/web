Formato de datos
El ingreso de datos se realiza a través de una planilla de cálculo con formato excel.
Para esto se requiere la utilización de tres hojas, en la primera se almacenan los datos propios de los árboles (arboles ) de cada parcela, en una segunda hoja se establece la lista de especies presentes (cod especies), las funciones de volumen del fuste.La tercer hoja utilizada describe los estados de los árboles (cod estados).
Para que el procesamiento se realice correctamente es necesario que las hojas del archivo excel sean denominadas como aquí se describe, la utilización de otro nombre no será reconocido por la calcuLEMUs. En próximas actualizaciones está limitación se removerá tal como sucede con el procesamiento de inventario de plantaciones. Para la lectura de los datos se emplean los primeros 5000 registros de la hoja arboles, es decir el encabezado y 4999 líneas con datos. El procesamiento verificará que al menos existan las primeras 20 columnas que a continuación se describen caso contrario el procesamiento se detiene.
